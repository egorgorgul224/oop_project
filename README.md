## Описание<a id="description"></a>
Проект представляет собой изучение ООП. В данном проекте реализовано два класса *Product* и *Category*. В классах
описаны атрибуты, реализована инициализация экземпляров класса. Для каждого класса написаны тесты.

## Установка и настройка проекта<a id="instruction"></a>

1. Клонируйте репозиторий:
```
git clone https://github.com/username/project-x.git
```
2. Перейдите в директорию проекта:
```
cd ваш_проект
```
3. Установите зависимости проекта:
```

```

## Функции, примеры их использования и тестирование<a id="func"></a>
1. Функция *get_excel_df* принимает на вход название excel файла и возвращает табличный формат файла(DataFrame).
Если название файла не указано, то принимает excel файл "operations.xlsx" в папке data. Если файл не был найден,
то вызывает исключение с сообщением "Файл не найден".

```
Вызов функции: get_excel_file("operations.xlsx")
Результат: DataFrame({}, {}, {}...)
```
Функция протестирована в модуле *test_file_reader* в папке **tests**. Также у функции обработано логирование в файл
*file_reader.log* в папке **logs**.

---

### Функция main для запуска и проверки проекта<a id="funcmain"></a>
1. Функция *main* возвращает приветствие, данные о сумме расходов и кэшбеку по каждой карте в периоде с начала месяца
    по дату пользователя(месяц берется из даты пользователя), топ 5 транзакций по сумме платежа, курс валют,
    стоимость акций. Также возвращает категории и кэшбек по ним за выбранный месяц и траты по выбранной категории
    за период 3 месяца.
